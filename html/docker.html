Docker

  • docker pull - получение образа приложения.
  • docker images - вывод списка образов.
  • docker run -dp - запуск контейнера с образом.
  • docker ps - вывод запущенных контейнеров.
  • docker stop id контейнера - остановка контейнера.
  • docker restart id контенера - перезапуск контейнера.
  • docker image rm -f id образа - удаление образа.
